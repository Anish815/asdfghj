Git Commands

//Use these commands in order to save

1) git init //Initialize Git
2) git add . //Prepare all files to be committed
3) git commit -m "*message*" //Saves the version of the files and prepares the files to be sent to GitHub 
4) git push *url* *branch* //Saves your files on GitHub (If you didn't create a branch for yourself, use "git branch")

Other Git commands

git clone *url* //Puts the repo onto your laptop, in a folder 
git branch *branch* //Creates a new branch (Don't do this if you have your own branch).
git pull *url* //Updates your folder with the repo
